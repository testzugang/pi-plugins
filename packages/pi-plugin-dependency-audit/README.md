# @testzugang/pi-plugin-dependency-audit

Static dependency and supply-chain malware auditing skill for Pi.

## Install

```bash
pi install npm:@testzugang/pi-plugin-dependency-audit
```

## Usage

```text
/skill:dependency-audit
```
