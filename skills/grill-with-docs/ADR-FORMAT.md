# ADR Format

Use this format when creating Architectural Decision Records.

```markdown
# ADR [Number]: [Title]

- **Date**: [YYYY-MM-DD]
- **Status**: Proposed | Accepted | Superseded
- **Context**: What is the problem we are solving? What are the constraints?
- **Decision**: What did we decide to do?
- **Consequences**: What is the impact of this decision? (Positive and negative)
```
